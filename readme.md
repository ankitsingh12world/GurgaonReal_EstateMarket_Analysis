# Gurgaon Real Estate Market Analysis

## Project Overview

This project focuses on analyzing residential property listings in Gurgaon using Exploratory Data Analysis (EDA). The objective is to generate data-driven insights that help buyers, investors, and developers make informed real estate decisions.

Using Python, Pandas, Matplotlib, and Seaborn, the project explores property pricing patterns, locality trends, builder influence, property types, and RERA approval impact.

---

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Jupyter Notebook

---

## Dataset

**Source:** Gurgaon Real Estate Dataset (Kaggle)

The dataset contains:

* Property Price
* Area (sqft)
* BHK Configuration
* Property Type
* Builder Details
* Locality
* Property Status
* RERA Approval Status

---

## Project Objectives

The analysis answers the following business questions:

1. Which is the costliest property?
2. Which locality has the highest average price?
3. Which locality has the highest rate per square foot?
4. Do ready-to-move properties cost more than under-construction properties?
5. Do RERA-approved properties command a price premium?
6. How does area impact property price?
7. Which BHK configuration is the most expensive?
8. Which property type is the costliest?
9. Do certain builders consistently price higher?
10. Are larger homes always more expensive per square foot?

## Exploratory Data Analysis (EDA)

### Property Pricing Analysis

* Identified the most expensive property in the dataset.
* Compared average property prices across localities.
* Evaluated price-per-square-foot trends.

### Property Characteristics Analysis

* Analyzed pricing differences across BHK configurations.
* Compared Apartments, Floors, and Plots.
* Studied the impact of property size on pricing.

### Market Factors Analysis

* Examined the effect of RERA approval on prices.
* Compared ready-to-move and under-construction properties.
* Evaluated builder reputation and premium pricing.

### Visualization

Created visualizations using Matplotlib and Seaborn:

* Scatter Plots
* Distribution Analysis
* Locality Comparisons
* Price Trend Analysis

---

## Key Insights

* Premium localities command significantly higher property prices.
* Higher rate-per-square-foot areas indicate strong market demand.
* RERA-approved properties generally attract premium pricing.
* Ready-to-move properties often cost more than under-construction properties.
* Builder reputation has a strong influence on property valuation.
* Larger homes are not always more expensive on a per-square-foot basis.
* Apartments generally have a higher rate per square foot compared to plots and floors.
